# 개인블로그 리뉴얼 프로젝트 (Frontend)

## 기본 정보

- 작업기간 : 2022.09.10. ~
- 어플리케이션 명 : selosele2
- Backend : Nest.js
- Frontend : Vue
- DBMS : MariaDB (& TypeORM)
- Port: 8080
- 프론트엔드 구동 : ```npm run serve``` &rarr; ```npm run sass:watch```
- 프론트엔드 빌드 : ```npm run build```
